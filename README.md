## 헬다이버즈2 스트라타젬 보조 기구
* **스트라타젬 단축키 자동 인식**<br>별도의 복잡한 설정 없이, 인게임 설정을 자동으로 인식합니다.

* **사후 데이터 관리**<br>업데이트가 중단되어도 `stratagems.json` 수정을 통해 스트라타젬을 추가할 수 있습니다.

* **라디얼 오버레이**<br>게임 내 오버레이로 직관적인 선택이 가능합니다.

* **슬롯 단축키**<br>슬롯에 지정된 스트라타젬을 바로 사용할 수 있습니다.

* **스트라타젬 자동 선택**<br>장비구성 메뉴에서 슬롯에 지정된 스트라타젬을 자동으로 선택합니다.

* [보조 기구 장착하기 (Download)](https://github.com/ChubbyMaru/HD2-Stratagem-Tool/releases/latest)

## 단축키 기본 값
* **증원 :** `마우스 버튼1(뒤로 가기)`
* **오버레이 :** `마우스 휠 클릭`
* **인게임 선택 :** `F1`
* **슬롯 :** `F6` ~ `F12`
